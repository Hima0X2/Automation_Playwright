## Project Description
This project automates a basic user flow on [SauceDemo](https://www.saucedemo.com/).  
The automated scenario includes:
1. Logging in with valid credentials  
2. Adding one product to the cart  
3. Check the product name in the cart  
4. Logging out  

## Prerequisites
Before running this project, make sure you have:
- **Node.js** installed (v16 or above recommended)  
- **npm** installed (comes with Node.js)  
- Internet access to reach SauceDemo  

## Project Setup
1. **Clone the repository**
```bash
git clone https://github.com/Hima0X2/Automation_Playwright
cd saucedemo
```
2. **Install dependencies**
```bash
npm install
```
3. **Install Playwright browsers**
```bash
npx playwright install
```
## Run Commands
1. **Run all tests**
```bash
npx playwright test
```
